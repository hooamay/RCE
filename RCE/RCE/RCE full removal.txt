Windows – Full Removal

:: 1. Remove persistence file
del "%APPDATA%\Microsoft\RCE.py"

:: 2. Remove registry startup entry
reg delete HKCU\Software\Microsoft\Windows\CurrentVersion\Run /v WinUpdater /f

:: 3. Remove keylogger output (if exists)
del "%USERPROFILE%\.keylog.txt"

:: 4. Optional: Clear evidence from Recent files (if run manually)
del /f /q "%APPDATA%\Microsoft\Recent\RCE.py.lnk"

Check for malicious running:
reg query HKCU\Software\Microsoft\Windows\CurrentVersion\Run


 Linux – Full Removal

# 1. Remove persistence file
rm ~/.config/.RCE.py

# 2. Remove autorun command from .bashrc (optional, safer to edit manually)
sed -i '/python3 .*\.RCE\.py/d' ~/.bashrc

# 3. Remove keylogger output (if exists)
rm ~/.keylog.txt

For both OS, clear temporary folders if your payload dropped screenshots or webcam images:

Windows:

del /f /q "%TEMP%\screen.png"
del /f /q "%TEMP%\webcam.jpg"
del /f /q "%TEMP%\LoginData.db"

Linux:

rm ~/screen.png ~/webcam.jpg ~/LoginData.db 2>/dev/null
